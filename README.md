## 🚀 바이브 코딩이란?
AI에게 자연어로 요구사항을 전달하고, 생성된 결과를 바탕으로 즉각적인 피드백을 주고받으며 프로그램을 완성해 나가는 **대화 중심의 개발 문화**를 의미합니다. 
기술적인 장벽에 막히지 않고, 오직 아이디어와 직관에 집중하여 결과물을 만들어내는 것이 목표입니다.
