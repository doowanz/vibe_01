# ✨ 바이브코딩 스터디 — 01회차

---

## 바이브 코딩이란?

> AI에게 자연어로 요구를 던지고, 결과를 보며 계속 수정·확장하는 **대화형 개발 방식**

---

## 🧠 핵심 가치

### 1. 속도 — Speed over Perfection
완벽한 설계보다 빠른 실행이 먼저입니다.
**"일단 만들어보자!"**

### 2. 대화 중심 개발 — Conversational Development
코드를 직접 쓰는 대신, **설명 → 생성 → 수정** 루프를 반복합니다.

**예시 대화 흐름**
```
"로그인 기능 만들어줘"
  → "에러 처리 추가해줘"
    → "React로 바꿔줘"
```

### 3. 점진적 구체화 — Iterative Refinement
처음부터 완벽할 필요 없습니다.
계속 질문하고 대화하면서 결과물을 개선해 나갑니다.

### 4. 맥락 위임 — Delegation to AI
**How(어떻게 구현할까)** 는 AI에게 맡기고,
사람은 **What(무엇을 만들까)** 과 **Why(왜 필요한가)** 에 집중합니다.

### 5. 직관 기반 흐름 — Flow-driven
먼저 화면을 띄운 뒤 즉흥적으로 반응합니다.
> "어? 저 버튼이 너무 작네? 키워줘"


## 🛠️ 직접 해보자

### 환경 설정

**1. Node.js 설치**
1. [https://nodejs.org/ko/download](https://nodejs.org/ko/download) 접속
2. 하단 **Windows 설치 프로그램** 클릭 → 다운로드
3. 설치 마법사에서 계속 **다음** 클릭하여 설치 완료

**2. Claude Team 가입**
1. [https://claude.ai](https://claude.ai) 접속 → Team 플랜 가입
2. 가입 완료 후 로그인 상태 유지

**3. Claude Code 설치**
```bash
npm install -g @anthropic-ai/claude-code
```

**4. 작업 폴더 생성**
```
문서 > Vibe > study_01
```

---

### 첫 실습

**1단계. 폴더에서 터미널(프롬프트) 열기**
`study_01` 폴더 경로에서 터미널을 실행합니다.

**2단계. Claude Code 실행**
```bash
claude
```

**3단계. 나만의 프로그램 만들기**
자유롭게 아이디어를 골라 만들어 보세요.

| 예시 아이디어 |
|---|
| 웹 계산기 |
| 맛집 추천 사이트 |
| 개인 프로필 사이트 |
| 오셀로 게임 |

**4단계. 디자인 입히기**

1. [https://wordpress.com](https://wordpress.com) 접속 → **테마** 메뉴 접속
2. 마음에 드는 테마 **캡처**
3. 캡처 이미지를 `study_01` 폴더 안에 저장
4. Claude Code에서 아래와 같이 요청
```
@파일명 파일을 참고해서 디자인 변경해줘
```
> `@`를 입력하면 파일을 직접 선택할 수 있습니다.
